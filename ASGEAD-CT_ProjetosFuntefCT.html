Consultas Projetos Funtef com UTFPR-CT

Esta página é para consultas de arrecadação, sendo dos projetos com a FUNTEF ou de demais serviços, organizados por UGR da UTFPR-CT, dos remanejamentos relacionados às arrecadações, os valores disponíveis, entre outros.

Consultas Projetos FUNTEF CT por número

Esta consulta retorna os dados dos projetos da UTFPR com a FUNTEF pelo número, agilizando a consulta do Conveniar e verificando se o projeto esta na base de dados das consultas deste site

Consultas Projetos FUNTEF CT por departamento

Esta consulta retorna os dados dos projetos da UTFPR com a FUNTEF, por UGR, porém vigente em algum momento de 2025, devido alta quantidade de processos.

Consultas Projetos FUNTEF CT Vigentes

Esta consulta retorna os projetos da UTFPR com a FUNTEF vigentes numa data específica.

Consulta GRUs Arrecadadas por Departamento da UTFPR-CT

Esta consulta retorna as GRUs recolhidas na UTFPR-CT com base no número de referência informado na GRU, que é vinculado ao projeto ou serviço da UTFPR-CT e de suas UGRs.

Consulta Referências para GRUs

Esta consulta retorna as referências estabelecidas pela UTFPR/DIRPLAD-CT, caso queira estabelecer uma nova referência, utilizadas para organização das informações das arrecadações, entre em contato com o DEOFI-CT.

Consulta Valores Repassados de Recurso Próprio

Esta consulta retorna os valores repassados ou utilizados por UGR referente os valores arrecadados, podem não ser iguais aos do SIORG, caso uma parte seja um acordo ou valor alheio ao arrecadado pela UGR.

Consultas Projetos FUNTEF CT - IA - Instável

Esta consulta retorna os dados dos projetos da UTFPR com a FUNTEF, conforme templates pré definidos, individual ou lista de projetos com base em UGR, coordenador, vigência, valores, etc.
